# Decision Twin — ML Team
Machine Learning module for the Decision Twin platform focused on predictive modeling, simulation, optimization, and decision intelligence.
## Tech Stack
- Python
- scikit-learn
- pandas
- NumPy
- XGBoost
- LightGBM
- Optuna
- MLflow
- SHAP
